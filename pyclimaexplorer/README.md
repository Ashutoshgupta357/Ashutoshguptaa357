# 🌍 PyClimaExplorer
> Interactive climate data visualiser · Technex '26 · Hack-it-out

PyClimaExplorer is a Streamlit web dashboard that loads NetCDF climate datasets
(ERA5, CESM, or any `.nc` file) and lets you explore them through interactive
heatmaps, time-series plots, and comparison tools — no coding required.

---

## Features

| Feature | Description |
|---|---|
| 🗺️ **Global Heatmap** | Spatial distribution of any variable for a selected month/year |
| 📈 **Time-Series Plot** | Monthly values at a chosen lat/lon with trend line |
| 📊 **Anomaly Mode** | Toggle climatological anomaly (value minus baseline) |
| ⚖️ **Comparison Mode** | Side-by-side heatmaps and dual time-series for two years/locations |
| 📖 **Story Mode** | Five guided tours: El Niño, Arctic amplification, ITCZ, jet streams |
| 📤 **Upload your own data** | Drag & drop any `.nc` / `.nc4` file |
| 📉 **Distribution Histogram** | Value distribution at selected location |
| 🌐 **Global Mean Trend** | Latitude-weighted global mean with decadal trend rate |

## Quick start

### 1 — Clone / download

```bash
git clone <your-repo-url>
cd pyclimaexplorer
```

### 2 — Install dependencies

```bash
pip install -r requirements.txt
```

> Python 3.11+ recommended. Works on macOS, Linux, and Windows.

### 3 — Generate the demo dataset

```bash
python generate_data.py
```

This creates `data/climate_demo.nc` (~40 MB), a synthetic ERA5-style file with:
- **t2m** — 2 m Temperature (K)  
- **tp**  — Total Precipitation (mm/day)  
- **u10** — 10 m Wind Speed (m/s)  
- 408 monthly time steps: **January 1990 – December 2023**  
- 73 × 144 global grid (2.5° resolution)

### 4 — Run the app

```bash
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

## Using your own ERA5 data

1. Go to [https://cds.climate.copernicus.eu](https://cds.climate.copernicus.eu) (free account)
2. Navigate to **ERA5 monthly averaged data on single levels**
3. Select variables: `2m temperature`, `total precipitation`, `10m u-component of wind`
4. Download as **NetCDF** (`.nc`)
5. In the sidebar, choose **"Upload your own .nc file"** and drag it in

The app auto-detects variable names and coordinate dimensions.

## Deploy to Streamlit Cloud

1. Push this repo to GitHub (include `requirements.txt`)
2. Go to [https://share.streamlit.io](https://share.streamlit.io)
3. Connect your repo and set `app.py` as the entry point
4. Add `data/climate_demo.nc` or configure file upload for users

> For cloud deploy, pre-generate `climate_demo.nc` and commit it, or add a
> data download step to the app startup.
